# Today API

## Responsibilities

- Read today's world mood and story
- Read daily recommendation
- Read public today summary

## Current contract

Today is a world-generated daily state, not a task system and not a签到 system.

## Reserved endpoint categories

- Today summary
- Today story
- Today mood
- Today news
- Today recommendation

## Rules

- Today content must feel like the world is speaking to the player.
- Do not turn Today into daily pressure or daily tasks.
