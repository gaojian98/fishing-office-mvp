# Milestone 1 Founder Bug Template

## Bug 反馈字段

- 页面
- 操作步骤
- 实际结果
- 期望结果
- 截图
- 严重程度

## 填写示例

**页面**
Home

**操作步骤**
打开首页 -> 点击抛线 -> 点击收线

**实际结果**
结果弹窗未出现，页面停在等待状态

**期望结果**
正常显示 FishResult，并可继续出售 / 留下 / 作为鱼饵

**截图**
请附上手机截图

**严重程度**
P1

## 严重程度建议

- `P0`：崩溃、白屏、无法进入
- `P1`：核心流程阻断
- `P2`：可继续但体验明显异常
- `P3`：轻微文案或视觉问题
